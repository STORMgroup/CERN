# Testing CERN

 - [./data](./data/) contains the datasets used in the CERN paper and their full versions, as well as scripts to fully process datasets for read-mapping. To run these scripts, install the necessary tools first in [./tools](./tools/)
 - [./evaluation](./evaluation) contains the scripts to run Rawhash2 on each of the datasets used in the paper, and summarize the results.
 - [./scripts](./scripts) contains useful scripts for running tools and processing files.
 - [./tools](./tools/) contains instructions to install the tools needed for processing datasets and generating results.
